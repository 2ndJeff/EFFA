# EFFA Website Redesign - File Package

## Contents

This folder contains all files for the EFFA website redesign project.

### HTML Files
- `index.html` - Homepage
- `effa-employers-page.html` - Employers page (For Employers)

### Image Files
- `effa-logo.png` - Main EFFA logo (color version)
- `effa-logo-white.png` - White version of logo (for dark backgrounds)
- `hero-laptop.png` - Hero section laptop image
- `step1-quick-setup.png` - Step 1 platform screenshot
- `step2-employee-enrollment.png` - Step 2 platform screenshot
- `step3-automated-compliance.png` - Step 3 platform screenshot
- `step4-simple-billing.png` - Step 4 platform screenshot

## How to View

1. Download this entire folder
2. Keep all files in the same folder (HTML files reference images by filename)
3. Open `index.html` or `effa-employers-page.html` in your web browser
4. All images should display properly

## Navigation

The navigation links between pages are set up as follows:
- Homepage: `index.html`
- Employers Page: `effa-employers-page.html`
- Other pages (Universities, About, Podcast, Contact) are not yet built
